# GlobalTech Sales Analysis

This repository contains a Python-based data analysis project using Pandas to analyze quarterly sales data for a multinational electronics retailer.

## Overview

The project focuses on analyzing structured sales data to identify trends, evaluate performance across regions and product categories, and generate business insights. It simulates how data analysts use Pandas to support decision-making in a real business environment.

## Features

* Load and explore sales data using Pandas
* Perform data filtering and selection using indexing
* Calculate key performance metrics such as total revenue and units sold
* Analyze sales by region, category, and product
* Evaluate promotion effectiveness
* Detect and analyze missing data
* Generate insights and business recommendations

## Files Included

* `module09-assignment.py` – Main Python file containing the full sales analysis
